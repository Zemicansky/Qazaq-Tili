# Qazaq Tili — Тренажёр казахского языка

Веб-приложение для изучения казахского языка: словарь по темам, тренировка слов, диалоги, статистика прогресса.

## Запуск

Просто откройте `index.html` в браузере — приложение полностью самодостаточное (без сервера и зависимостей), прогресс хранится в localStorage браузера.

## GitHub Pages

Сайт публикуется через GitHub Pages из этого репозитория (ветка `main`, корень `/`).
